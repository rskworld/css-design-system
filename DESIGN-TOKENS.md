CSS custom properties defined in `css/tokens.css`. Override in your stylesheet or via `data-theme="dark"` (see `themes.css`).

## Colors

| Token | Default | Description |
|-------|---------|-------------|
| `--rsk-color-primary` | #4a90e2 | Primary brand |
| `--rsk-color-primary-light` | #6ba3e8 | Lighter primary |
| `--rsk-color-primary-dark` | #357abd | Darker primary |
| `--rsk-color-primary-contrast` | #ffffff | Text on primary |
| `--rsk-color-success` | #28a745 | Success state |
| `--rsk-color-warning` | #ffc107 | Warning state |
| `--rsk-color-danger` | #dc3545 | Error/danger |
| `--rsk-color-info` | #17a2b8 | Info state |
| `--rsk-color-gray-50` … `--rsk-color-gray-900` | — | Neutral scale |
| `--rsk-color-white`, `--rsk-color-black` | #fff, #000 | Base |

## Typography

- **Fonts:** `--rsk-font-sans`, `--rsk-font-mono`, `--rsk-font-serif`
- **Sizes:** `--rsk-text-xs` (0.75rem) … `--rsk-text-5xl` (3rem)
- **Weights:** `--rsk-font-normal` (400), `--rsk-font-medium` (500), `--rsk-font-semibold` (600), `--rsk-font-bold` (700)

## Spacing (4px base)

`--rsk-space-0` … `--rsk-space-24` (e.g. `--rsk-space-4` = 1rem)

## Radius

`--rsk-radius-sm` (0.25rem) … `--rsk-radius-full` (9999px)

## Shadows

`--rsk-shadow-sm`, `--rsk-shadow-md`, `--rsk-shadow-lg`, `--rsk-shadow-xl`

## Motion

- **Duration:** `--rsk-duration-fast` (150ms), `--rsk-duration-normal` (300ms), `--rsk-duration-slow` (500ms)
- **Easing:** `--rsk-ease-default`, `--rsk-ease-in`, `--rsk-ease-out`

## Breakpoints (reference)

- `--rsk-bp-sm`: 36rem (576px)
- `--rsk-bp-md`: 48rem (768px)
- `--rsk-bp-lg`: 62rem (992px)
- `--rsk-bp-xl`: 75rem (1200px)
- `--rsk-bp-2xl`: 87.5rem (1400px)

## Container max-widths

`--rsk-container-sm` (540px) … `--rsk-container-2xl` (1320px)
