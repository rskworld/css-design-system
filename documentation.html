Design Tokens

CSS custom properties in css/tokens.css. Colors, typography, spacing, radius, shadows, breakpoints.

/* Example usage */
color: var(--rsk-color-primary);
font-size: var(--rsk-text-lg);
padding: var(--rsk-space-4);
border-radius: var(--rsk-radius-lg);
box-shadow: var(--rsk-shadow-md);

Typography

css/typography.css. Headings .rsk-h1–.rsk-h6, .rsk-lead, .rsk-blockquote, .rsk-code, .rsk-pre, .rsk-kbd, lists (unstyled, inline, styled), .rsk-dl-inline. See examples.html.

Layout

css/layout.css. Stack .rsk-stack-1–.rsk-stack-6, cluster .rsk-cluster, sidebar .rsk-with-sidebar, split .rsk-split, center .rsk-center, flow .rsk-flow, region .rsk-region, wrapper .rsk-wrapper, box .rsk-box.

Forms advanced

css/forms-advanced.css. Form group .rsk-form-group, hint/error, checkbox/radio .rsk-form-check, switch .rsk-switch (track + thumb), input group .rsk-input-group + addon, validation .is-valid / .is-invalid.

Tables

css/tables.css. .rsk-table, .rsk-table-striped, .rsk-table-bordered, .rsk-table-compact, .rsk-table-responsive (scroll wrapper).

Effects

css/effects.css. Progress .rsk-progress + .rsk-progress-bar, avatar .rsk-avatar (sm/lg/xl), .rsk-avatar-group, divider .rsk-divider / .rsk-divider-text, spinner .rsk-spinner, empty state .rsk-empty-state, .rsk-card-img-top, blur .rsk-blur.

Callouts

css/callouts.css. Highlight boxes: .rsk-callout with .rsk-callout-info, .rsk-callout-success, .rsk-callout-warning, .rsk-callout-danger. Optional .rsk-callout-icon, .rsk-callout-title, .rsk-callout-content. Dark theme supported.

Scroll

css/scroll.css. Scroll-snap: .rsk-scroll-snap-x (horizontal carousel), .rsk-scroll-snap-y (vertical sections). .rsk-scroll-margin-header for anchor offset. Scrollbar: .rsk-scrollbar-thin, .rsk-scrollbar-hide. .rsk-scroll-smooth (respects reduced motion).

Timeline & Steps

css/timeline.css. Vertical timeline: .rsk-timeline, .rsk-timeline-item (optional .rsk-timeline-item-success / -warning / -danger), .rsk-timeline-time, .rsk-timeline-title, .rsk-timeline-content. Horizontal stepper: .rsk-steps, .rsk-step with .is-active / .is-done, .rsk-step-number, .rsk-step-label.

Overlays

css/overlays.css. Card overlay: .rsk-card-overlay, .rsk-card-overlay-img, .rsk-card-overlay-caption. Hero: .rsk-hero-overlay, .rsk-hero-overlay-content. Backdrop: .rsk-overlay-backdrop. Media caption: .rsk-media-overlay, .rsk-media-overlay-caption.

Selection & Sticky

css/selection.css. Custom ::selection (primary tint). Sticky: .rsk-sticky-top, .rsk-sticky-bottom, .rsk-sticky-top-2, .rsk-sticky-top-4. Z-index: .rsk-z-0–.rsk-z-50, .rsk-z-dropdown, .rsk-z-modal, .rsk-z-tooltip.

Grid System

12-column flexbox grid. Use .rsk-container, .rsk-row, and .rsk-col-* / .rsk-col-{sm|md|lg}-*.

<div class="rsk-container">
  <div class="rsk-row">
    <div class="rsk-col rsk-col-12 rsk-col-md-6">Column</div>
    <div class="rsk-col rsk-col-12 rsk-col-md-6">Column</div>
  </div>
</div>

Utility Classes

Display, flex, spacing, text, width, visibility, border, shadow. Prefix: rsk-.

Examples: .rsk-d-flex, .rsk-justify-center, .rsk-m-4, .rsk-p-3, .rsk-text-center, .rsk-font-bold, .rsk-rounded.

Components

Buttons (.rsk-btn, .rsk-btn-primary, .rsk-btn-outline-primary), cards (.rsk-card, .rsk-card-header, .rsk-card-body, .rsk-card-footer), badges (.rsk-badge), alerts (.rsk-alert), form inputs (.rsk-input, .rsk-select, .rsk-textarea, .rsk-label), nav (.rsk-nav, .rsk-nav-link), list group (.rsk-list-group, .rsk-list-group-item).

Themes

css/themes.css. Light/dark via data-theme="light" or data-theme="dark". Semantic tokens: --rsk-surface, --rsk-text, --rsk-border. Respects prefers-color-scheme: dark when no theme set.

Animations

css/animations.css. Keyframes: fade-in, slide-up, scale-in, spin, pulse, bounce, shimmer. Classes: .rsk-animate-fade-in, .rsk-animate-pulse, .rsk-transition, .rsk-skeleton. Respects prefers-reduced-motion.

Advanced Components

css/components-advanced.css. Modal (backdrop + .rsk-modal), Tabs (.rsk-tabs, .rsk-tabs-trigger, .rsk-tabs-panel), Dropdown (.rsk-dropdown, .rsk-dropdown-menu), Tooltip (.rsk-tooltip, .rsk-tooltip-content), Breadcrumb, Pagination, Accordion. See components.html for demos.

Focus & Print

css/focus.css – :focus-visible ring on interactive elements, .rsk-skip-link. css/print.css – print media query: hide modals/dropdowns, show link URLs, avoid breaks inside cards.

Responsive Patterns

.rsk-show-sm / .rsk-hide-sm (and md, lg), .rsk-responsive-text, .rsk-responsive-heading, .rsk-responsive-grid, .rsk-stack / .rsk-stack-row, .rsk-img-fluid, aspect ratios. Respects prefers-reduced-motion.
