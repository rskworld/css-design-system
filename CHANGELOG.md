## [1.0.0] - 2026-02-08

### Added

- **Core:** Design tokens (`tokens.css`), themes (`themes.css`), grid (`grid.css`), utilities (`utilities.css`).
- **Typography:** Headings, lead, blockquote, code/pre, kbd, lists (`typography.css`).
- **Layout:** Stack, cluster, sidebar, split, center, flow, region, wrapper, box (`layout.css`).
- **Components:** Buttons, cards, badges, alerts, form inputs, nav, list group (`components.css`).
- **Advanced components:** Modal, tabs, dropdown, tooltip, breadcrumb, pagination, accordion (`components-advanced.css`).
- **Forms advanced:** Form group, checkbox/radio, switch, input group, validation states (`forms-advanced.css`).
- **Tables:** Table, striped, bordered, compact, responsive (`tables.css`).
- **Effects:** Progress, avatar, divider, spinner, empty state, blur (`effects.css`).
- **Callouts:** Info, success, warning, danger callouts (`callouts.css`).
- **Scroll:** Scroll-snap (x/y), scrollbar thin/hide, scroll-margin, smooth scroll (`scroll.css`).
- **Timeline:** Vertical timeline, horizontal stepper (`timeline.css`).
- **Overlays:** Card overlay, hero overlay, backdrop, media caption (`overlays.css`).
- **Selection & sticky:** Custom selection, sticky utilities, z-index scale (`selection.css`).
- **Animations:** Keyframes, animation/transition utilities, skeleton (`animations.css`).
- **Focus:** Focus-visible ring, skip link (`focus.css`).
- **Responsive:** Visibility by breakpoint, responsive grid, reduced motion (`responsive.css`).
- **Print:** Print styles (`print.css`).
- **Pages:** `index.html`, `components.html`, `examples.html`, `documentation.html`.
- **Docs:** README, USAGE, CONTRIBUTING, CHANGELOG, CODE_OF_CONDUCT, LICENSE.
